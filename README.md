# Android-Storage-Extractor
*v0.9*

A tool to extract *local data storage* of an Android application in one **__click__**.

***
Usage
------------------

> python extract.py
>
> __Options__:
>
> -p    -package name
>
> -o    -output directory
>
> -h    -help

***

Run
------------------

> __Wizard mode__:
>
> python extract.py

> __Command line__:
>
> python extract.py -p com.android.chrome -o /root/Desktop/  

> python extract.py -p com.android.chrome
